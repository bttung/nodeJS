

/search.html にアクセスされたらテキストボックスとサブミットボタンのフォームを設置したHTMLをレスポンスする。

/search.html?word=xx というURLにGETでアクセスされたら、
(元の)テキストボックス、サブミットボタンの下に、以下のように日本語訳したワードを表示する。
(左が入力されたら、右を表示する)
テキストボックスには入力されたワードを再表示。


Hello -> こんにちは
Bye -> さようなら
Hey -> へい
Hi -> やあ


他の単語は、

xxは辞書に登録されていません

と表示する。
大文字小文字は区別しない、ワード前後のスペースは無視する

******************************************************

/search.txt?word=xx というURLにGETでアクセスされたら、
日本語訳の結果を plain text としてレスポンスする。
ない単語は ステータス404 をレスポンスする。

/search.txt?word=xx&word=yy というように複数ワードの場合は、
最初のワードの日本語訳の結果を plain text としてレスポンスする。

/search.json?word=xx というURLにGETでアクセスされたら、
{ "xx": "ほげ" } というJSONフォーマットをレスポンスする。
ない単語は { "xx": null } というように null をセットする。


/search.json?word=xx&word=yy というように複数ワードの場合は、
{ "xx": "ほげ", "yy": "ふが" } というJSONフォーマットをレスポンスする。
ない単語は上記と同様。
